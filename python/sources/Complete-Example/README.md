# Python example

[This project](https://github.com/quixio/quix-samples/tree/main/python/sources/Complete-Example) demonstrates how to connect to Kafka and publish data to it using [QuixStreams](https://quix.io/docs/client-library-intro.html).

## How to run

Create a [Quix](https://portal.platform.quix.ai/self-sign-up?xlink=github) account or log-in and visit the Samples to use this project.

Clicking `Edit code` on the Sample, forks the project to your own Git repo so you can customize it before deploying.

## Environment variables

The code sample uses the following environment variables:

- **output**: Name of the output topic to write into.
