# Currency exchange rate data

[This project](https://github.com/quixio/quix-samples/tree/main/python/sources/Currency-Exchange-Rates) replays currency exchange rates from September 2021 as if they happened today.

This example is intended to provide some data for you to experiment with.

## How to run

Create a [Quix](https://portal.platform.quix.ai/self-sign-up?xlink=github) account or log-in and visit the Samples to use this project.

Clicking `Deploy` on the Sample, deploys a pre-built container in Quix. Complete the environment variables to configure the container.

Clicking `Edit code` on the Sample, forks the project to your own Git repo so you can customize it before deploying.

## Environment variables

The code sample uses the following environment variables:

- **output**: This is the output topic for the data.
