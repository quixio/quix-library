# Empty template

[This project](https://github.com/quixio/quix-library/tree/main/python/destinations/Empty-Template-Panda) is an example on how to connect to, and handle the arrival of, a new stream and its data using Panda Data Frames.

## How to run

Create a [Quix](https://portal.platform.quix.ai/self-sign-up?xlink=github) account or log-in and visit the Library to use this project.

Clicking `Edit code` on the library item, forks the project to your own Git repo so you can customize it before deploying.

## Environment variables

The code sample uses the following environment variables:

- **input**: This is the input topic for demo data.
